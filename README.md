# Quizi_v2

Quizi_v2 is a modern React-based quiz application built with Vite. It provides a fast, responsive, and interactive user experience for creating and taking quizzes.

## 🚀 Features

- ⚡ Built with Vite for fast development and optimized builds
- ⚛️ React-based UI for dynamic and interactive components
- 🧠 Quiz system for testing knowledge
- 🎯 Real-time feedback on answers
- 📱 Responsive design for all devices
- 🧹 Code linting for better code quality

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- HTML5 & CSS3
- ESLint

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/imon025/Quizi_V2.git
cd quizi_v2
```

Install dependencies:

```bash
npm install
```

## ▶️ Available Scripts

### Start Development Server

```bash
npm run dev
```

Runs the app in development mode.  
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build for Production

```bash
npm run build
```

Creates an optimized production build in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

Locally preview the production build.

### Run Linter

```bash
npm run lint
```

Checks code quality and highlights issues.
